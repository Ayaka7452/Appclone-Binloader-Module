# Appclone-Binloader-Module
a binloader module which could install dual app to the space of specified android user id
# Usage
1. Flash it via binloader
2. Use android terminal and type: appclone [PACKAGE_NAME] (root required)
3. Go to moddir (/data/binloader/mods/App_Clone), edit defusr.cfg and apply the user id you wanted to install apps to (eg. 95)
5. Wait for installation, when it returns 'Success', the dual app icon will appear in your desktop
# Requirement
Android 13+ (only tested at 13), Magisk installed and Binloader installed
